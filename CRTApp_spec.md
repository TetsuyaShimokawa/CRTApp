# CRTApp Web版 仕様書

## 概要
認知反射テスト（CRT: Cognitive Reflection Test）のWebアプリケーション。
被験者がランダムな順序で10問の数学的・論理的問題に回答する。
回答は自由入力（数値テキスト）形式。

## 技術スタック
- **フロントエンド**: React (Vite)
- **バックエンド**: FastAPI (Python)
- **デプロイ**: Render

---

## 画面構成

### 画面1：セットアップ画面（Setup）
- 「Input your ID」ラベル
- 被験者IDの入力欄（テキスト入力）
- Startボタン
- バリデーション：IDが未入力の場合エラーメッセージを表示

### 画面2：質問画面（Question）
- 上部：問題文（フォントサイズ14px以上・複数行対応）
- 中央：回答入力欄（テキストエリア・半角数字）
- 中央左：赤字で「回答（半角数字）」ラベル
- 中央右：単位ラベル（例：「円」「分」「日」）
- 右下：「次へ」ボタン

### 画面3：終了画面（Finish）
- 「実験は終了です。お疲れ様でした。」メッセージ
- CSVダウンロードボタン

## 実験ロジック

### ランダム化
- **問題の提示順序**：全10問をランダムに並び替えて提示

### 回答方式
- テキストエリアに半角数字を入力
- 「次へ」ボタンで確定・次の問題へ
- 回答後テキストエリアをクリア

### 正誤判定
- フロントエンドでは判定しない
- バックエンドで保存のみ（正答もあわせて保存）

## データ記録（1行 = 1回答）

| カラム名 | 型 | 内容 |
|---------|-----|------|
| SubjectID | string | 被験者ID |
| QuestionIndex | integer | 元の問題番号（ランダム化前のID） |
| Response | string | 被験者の回答（入力テキスト） |
| CorrectAnswer | string | 正答 |

## UIデザイン詳細

### 問題文
- 複数行テキスト対応（改行を含む問題あり）
- フォントサイズ：14px以上
- 上部に大きく表示

### 回答エリア
- 赤字ラベル「回答（半角数字）」
- テキスト入力欄（1行）
- 単位ラベル（問題によって異なる：円・分・日・人・ドル・位・匹・立方フィート・空白）

### 「次へ」ボタン
- 右下に配置
- 回答入力後は常に押せる（バリデーションなし）

## バックエンドAPI（FastAPI）

### エンドポイント

#### `GET /api/questions`
ランダム順序の全問題データを返す

**レスポンス例：**
```json
[
  {
    "id": 3,
    "text": "ある湖にスイレンの葉が浮かんでいます...",
    "unit": "日",
    "correct_answer": "47"
  }
]
```

#### `POST /api/results`
回答を1件保存

**リクエスト例：**
```json
{
  "subject_id": "ABC123",
  "question_index": 3,
  "response": "47",
  "correct_answer": "47"
}
```

#### `GET /api/results/{subject_id}/csv`
CSVダウンロード
ファイル名：`AnsCRT_{subject_id}_{YYYYMMDD_HHMMSS}.csv`

## KnowledgeAssessmentAppとの違い

| 項目 | KnowledgeAssessmentApp | CRTApp |
|------|----------------------|--------|
| 回答方式 | ラジオボタン選択 | **テキスト自由入力** |
| 問題順序 | 固定 | **ランダム** |
| 利得行列 | あり（一部） | **なし** |
| 単位表示 | なし | **あり（問題ごとに異なる）** |
| 正答保存 | なし | **あり** |
| ファイル名 | questionnaire_... | **AnsCRT_...** |
